# RWB---WildHack
Решение хакатона от RWB (Объединённая компания Wildberries и Russ) по прогнозированию объёма отгрузок по логистическим маршрутам
Репозиторий содержит решение задачи прогнозирования объёма отгрузок (target_1h) по логистическим маршрутам (route_id) на основе исторических данных.

Модель использует подход direct multi-horizon forecasting и алгоритм CatBoost, а также расширенную генерацию признаков временного ряда.

## Постановка задачи
Разработать модель прогнозирования временного ряда, которая поможет спрогнозировать объёмы отгрузок и обеспечить стабильную работу складов в пиковые периоды
В тестовом наборе отсутствуют текущие признаки статусов обработки товаров, поэтому модель должна использовать только историческую информацию.

## Цель
Предсказать количество товаров, которые будут отгружены по каждому маршруту в будущие временные моменты.

Метрика хакатона:
WAPE + Relative Bias

## Подход к решению
Решение построено как задача многогоризонтного прогнозирования временных рядов (multi-horizon forecasting). Вместо последовательного (recursive) прогнозирования используется стратегия direct forecasting, при которой обучается отдельная модель для каждого горизонта прогноза. Это позволяет уменьшить накопление ошибки и учитывать различия в динамике краткосрочных и более удалённых прогнозов.

В качестве базового алгоритма выбран CatBoostRegressor, так как он хорошо работает с табличными данными, устойчив к шуму и способен эффективно моделировать нелинейные зависимости между признаками. 

Для каждого горизонта прогноза формируется отдельная обучающая выборка, в которой в качестве целевой переменной используется значение ```target_1h```, сдвинутое на соответствующее количество временных шагов вперёд. Таким образом, модель обучается напрямую предсказывать будущее значение.
